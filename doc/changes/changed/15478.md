- `dune describe workspace` now reports an `origin` field for every module,
  indicating whether its source was written by hand (`source`), generated by a
  `rule` stanza (`dune`, with the generating dune file and the rule's
  dependencies), produced by `ocamllex`, `ocamlyacc`, or `menhir`, or
  synthesised by dune itself for wrapping (`wrapper`). Generated modules point
  at the responsible file.
  (#15478, @nat-karmios)
